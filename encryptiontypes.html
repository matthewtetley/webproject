Public Key

There are currenlty many encryption types that are currently used, however the most popular in cloud soltuions are Public Key and Private Key while there are various Encrytion Standards in use for securing data.

Public Key Relies on someone having a document or file which when sent through the service uses a public key, encrypting the contents in the process. The Public key contains the result of 2 big prime numbers multiplied together.

Private Key

The reciever on the other side uses their private key to decrypt and read the contents of the file or document. The Private Key is the result of adding the prime numbers.

AES and Other Standards

There are currently a range of encryption standards and protocols that are used within filing systems but some of the current ones in use are AES, RSA, 3DES and Twofish which uses different types of ciphers and are generally quite secure.

The Level of Complexity can depend on what Encryption standard is being used and for filing systems with passwords protecting it, the complexity of the passsword. Windows 10 for Example uses a version of RSA utilising public and private keys.
